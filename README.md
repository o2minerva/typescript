## Prerequisites

- [Node = v10.15.x](https://nodejs.org/en/)
- NPM >= v6.4.x
- [Yarn >= v1.22.0](https://yarnpkg.com/en/docs/install#linux-tab) or `npm install -g yarn`

## Usage

**compile all files:**

```
yarn build
```

**compile and run a single file:**

```
yarn start classes/basic
```

**clean:**

```
yarn clean
```
